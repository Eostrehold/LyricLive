# LyricLive

一个基于 Fabric 26.1.2 的纯客户端模组，用于在 Minecraft 游戏内举办唱歌比赛或演唱会时，为玩家提供歌词显示与歌词发送功能。

## 功能特性

### 1. 歌曲时间控制
- 支持手动设置歌曲播放起始时间
- 支持开始、暂停、继续、停止播放
- 支持调整播放进度，实现歌词同步

### 2. LRC 歌词导入
- 支持导入标准 .lrc 歌词文件
- 自动解析歌词时间轴与文本内容
- 支持重新加载或切换歌词文件

### 3. 图形化用户界面（GUI）
- 提供简洁易用的图形化界面
- 支持选择歌词文件
- 支持设置歌曲时间、播放控制及相关配置
- 提供歌词预览功能

### 4. 客户端歌词显示
- 根据 LRC 时间轴，在客户端屏幕指定位置实时显示歌词
- 支持歌词居中显示
- 支持自定义字体颜色、缩放、透明度及显示位置

### 5. 自动发送歌词
- 根据 LRC 时间轴，自动将当前歌词发送至游戏聊天栏
- 支持设置发送频道（聊天栏/指令等）
- 支持开启或关闭自动发送功能

### 6. 手动发送歌词
- 提供快捷键或 GUI 按钮
- 用户可随时发送当前歌词，而无需等待自动发送

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| L | 打开 LyricLive 界面 |
| P | 播放/暂停 |
| O | 停止 |
| K | 发送歌词 |
| J | 切换自动发送 |

## 安装

1. 确保已安装 Minecraft 26.1.2
2. 确保已安装 Fabric Loader 0.19.3 或更高版本
3. 下载 LyricLive 模组文件
4. 将模组文件放入 `.minecraft/mods` 文件夹
5. 启动 Minecraft

## 使用方法

### 加载歌词文件
1. 按 `L` 键打开 LyricLive 界面
2. 点击"加载歌词"按钮
3. 选择 .lrc 歌词文件

### 播放控制
- 按 `P` 键开始播放或暂停
- 按 `O` 键停止播放
- 在 GUI 中使用播放控制按钮

### 歌词显示
- 歌词会自动显示在屏幕上
- 可以在设置中调整显示位置、颜色、大小等

### 自动发送
- 按 `J` 键切换自动发送功能
- 启用后，歌词会自动发送到聊天栏

### 手动发送
- 按 `K` 键手动发送当前歌词

## 配置

### 显示配置
- X/Y 位置：控制歌词在屏幕上的位置（0.0-1.0）
- 字体大小：控制歌词字体大小（8-64）
- 字体颜色：设置歌词颜色（十六进制）
- 透明度：设置歌词透明度（0.0-1.0）
- 阴影：是否启用文字阴影
- 居中：是否居中显示歌词
- 渐变：是否启用淡入淡出效果

### 发送配置
- 聊天发送：启用/禁用自动发送到聊天栏
- 指令发送：启用/禁用自动发送指令
- 指令模板：自定义指令格式（{lyric} 会被替换为歌词文本）

## LRC 文件格式

LyricLive 支持标准 LRC 格式：

```
[ti:歌曲标题]
[ar:艺术家]
[al:专辑]
[by:歌词作者]

[00:00.000]第一行歌词
[00:05.000]第二行歌词
[00:10.000]第三行歌词
```

支持：
- 元数据标签：[ti:], [ar:], [al:], [by:]
- 时间标签：[mm:ss.SSS]
- 多个时间标签指向同一行歌词
